# Plataforma Waiwahost - Informe del Proyecto

## Descripción General
Plataforma Waiwahost es una solución integral (PMS - Property Management System) diseñada para la administración eficiente de alquileres vacacionales y gestión de propiedades. El sistema centraliza la operación diaria, desde la reserva hasta el control financiero, ofreciendo herramientas tanto para administradores como para propietarios y huéspedes.

## Funcionalidades Principales

### 1. Gestión de Reservas
Módulo central para el control total del ciclo de vida de las reservas.
*   **Calendario Interactivo**: Visualización de disponibilidad con filtro por ciudades, y creación/edición rápida de reservas directamente desde el grid.
*   **Gestión de Estados**: Flujo completo de estados (Pendiente, Confirmada, Cancelada, Finalizada).
*   **Plataformas de Origen**: Identificación del canal de venta (Airbnb, Booking.com, Directo, etc.) para análisis de rendimiento.
*   **Cálculo Automático**: Desglose automático de totales, pagos realizados y saldos pendientes.

### 2. Control Financiero (Caja)
Sistema robusto de contabilidad simplificada para propiedades.
*   **Movimientos**: Registro detallado de cada Ingreso y Egreso.
*   **Categorización**: Clasificación precisa de gastos (Limpieza, Mantenimiento, Servicios Públicos, Comisiones) e ingresos.
*   **Balance por Propiedad**: Visualización de la rentabilidad específica de cada inmueble.
*   **Pagos Parciales**: Posibilidad de registrar múltiples pagos abonados a una misma reserva.

### 3. Gestión de Inmuebles y Propietarios
Base de datos centralizada de activos.
*   **Ficha de Inmueble**: Información detallada de cada propiedad (Dirección, características).
*   **Asignación de Propietarios**: Vinculación clara entre propiedades y sus dueños reales para liquidaciones.

### 4. Check-in Digital (Formulario Público)
Herramienta para mejorar la experiencia del huésped y reducir carga operativa.
*   **Auto-rregistro**: Los huéspedes reciben un enlace para completar sus datos personales (Documento, Teléfono, Email) antes de la llegada.
*   **Sincronización**: Los datos ingresados se actualizan automáticamente en la reserva correspondiente dentro de la plataforma.

### 5. Reportes e Inteligencia
*   **Dashboard Financiero**: Gráficos y resumenes de ingresos vs egresos.
*   **Exportación**: Generación de reportes y comprobantes (PDF) para propietarios y administración.

### 6. Seguridad y Administración
*   **Login Seguro**: Autenticación robusta con correo y contraseña encriptada.
*   **Roles y Permisos**: Diferenciación de accesos según el perfil del usuario (Administrador, Staff, etc.).
*   **Multi-Empresa**: Arquitectura preparada para gestionar múltiples organizaciones o cuentas de administración en una sola instalación.
